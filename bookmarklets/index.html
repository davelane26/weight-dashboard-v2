Garmin Sync Auto-Update

Scrapes your currently-open Garmin Connect page (Activity or Sleep) and pushes the numbers to your Firebase dashboard. One click, no re-typing, no CSV exports.

Drag this to your bookmarks bar:

Garmin Sync

This loader fetches the latest code from GitHub Pages every time you click it. When you improve garmin-sync.js in the repo, the bookmarklet automatically uses the new version - no re-install needed.

Drag this to your bookmarks bar:

Garmin Sync (self-contained)

This version bundles all the code inside the bookmarklet URL. Bigger URL, but works even if GitHub Pages is down. Won't auto-update when you edit the source.

Before it works: Make sure your bookmarks bar is visible (Ctrl+Shift+B). Then drag the button above onto it. Chrome may prompt to confirm the javascript: URL - that's normal.

How to use

  1. Open connect.garmin.com and log in (if not already)
  2. Navigate to either the Activity dashboard (default landing page) or the Sleep page
  3. Click the Garmin Sync bookmarklet on your bookmarks bar
  4. You'll see an alert() with the scraped values
  5. Data is pushed to Firebase automatically - refresh your dashboard to see it

What it captures

Activity page
Steps, resting HR, active calories, total calories, intensity minutes, Body Battery, stress, fitness age
Sleep page
Sleep hours, sleep score, deep sleep, light sleep, REM sleep
Where it goes
Firebase /garmin/{today}.json and /garmin/latest.json, plus a backup POST to your Cloudflare Worker

Troubleshooting

  • Alert shows all "--" values - Garmin updated their UI and broke the DOM selectors. Inspect an element on connect.garmin.com and update garmin-sync.js.
  • Nothing happens on click - Chrome may have stripped the javascript: prefix when saving. Right-click the bookmark - Edit - make sure the URL still starts with javascript:.
  • "Not Logged In" in alert - Your Garmin session expired. Log back in on connect.garmin.com and try again.
  • Console shows "Sync error" - Firebase or the worker rejected the write. Check network tab for the specific status code.
